## Stratégie globale

**Objectif** : Notoriété → Démonstration → Conversion (essai gratuit)  
**Audience** : Commerçants, restaurateurs, prestataires de services, PME Sénégal/Afrique de l'Ouest  
**CTA unique** : elm-app.click (essai 7 jours gratuit)

**Logique de séquençage** :  
1. Commencer par les douleurs universelles (tout le monde se reconnaît)  
2. Enchaîner avec les preuves de valeur (WhatsApp, hors-ligne)  
3. Aller vers les niches métier (hôtel, garage, juridique...)  
4. Finir sur la puissance et la croissance (analytics, multi-sites)
